## 안녕하세요!

이 블로그는 **Next.js 14**와 **MDX**로 만들어진 터미널 스타일 블로그입니다.

### 주요 기능

개발자를 위한 깔끔한 터미널 스타일 디자인을 제공합니다:

- 마크다운 지원
- 코드 하이라이팅
- 다크 모드
- 반응형 디자인

### 코드 예제

터미널처럼 생긴 코드 블록도 지원합니다:

```typescript
function hello(name: string) {
  console.log(`Hello, ${name}!`);
  return `Welcome to my blog`;
}

hello("Developer");
```

### 인라인 코드

텍스트 중간에 `const greeting = "Hello"` 같은 인라인 코드도 사용할 수 있습니다.
